# Assets Directory

Thư mục này chứa các tài nguyên tĩnh cho EduManager Pro.

## Cấu trúc

```
assets/
├── images/          # Hình ảnh
│   ├── logo.png
│   ├── avatar.png
│   └── backgrounds/
├── icons/           # Icons
├── fonts/           # Custom fonts
└── documents/       # Tài liệu mẫu
```

## Hướng dẫn

### Logo
- Kích thước khuyến nghị: 512x512px
- Format: PNG với transparent background
- Đặt tại: `assets/logo.png`

### Avatar
- Kích thước: 150x150px
- Format: PNG hoặc JPG
- Đặt tại: `assets/avatar.png`

### Images
- Optimize trước khi upload
- Sử dụng WebP nếu có thể
- Đặt tên file rõ ràng, không dấu

## Placeholder Images

Hiện tại đang sử dụng placeholder từ:
- https://via.placeholder.com/

Thay thế bằng images thật khi production.
